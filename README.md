# RIME 諺文粵語方案 rime-jyuteul2set
基於rime平台使用諺文輸出粵文的輸入方案 Base Rime Use Hangul Output Cantonese Scheme

使用韓語諺文2set鍵盤方式輸出，形似三拼或四拼。方案非標準。參考來自於基於知乎用户[荷達-粵語諺文方案](https://www.zhihu.com/question/27563380/answer/123653538)。

優點在於：挪用gwㄲ kwㄸ  wㅆ zㅈ jㅉ ㅓa ㅔai ㅏaa ㅐai ㅕeo ㅑoe  即在範圍內字符情況下覆蓋粵文

## 特性&使用

分 `jyuteul2set(運算映射版)` 和 `jyuteul2setreal(真碼表版)`，前者可打常規粵語音節表內範圍，後者可混打諺文粵文。

反查使用，普通話 `w:` 兩分 `r:` 筆劃 `x:` 倉頡`v:` （注意有`:`號）。

輸入編碼後，後綴按\`鍵，可進入諺文優先模式。

 -  下載安裝Rime，下載本方案，部署到“用户文件夾”，“輸入法設定”選中本方案，部署即可
